### 常用命令
```sh
#修改设备名称
sysname name1
#关闭提示信息
undo info-center enable
#查看设备ip地址与设备接口
display ip interface brief
#查看VLAN
display vlan
#查看端口状态
display interface brief
#查看全部配制信息
display current-configuration
```
